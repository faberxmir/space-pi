# space-pi
Interactive pi-spaceship for used in 2ITA wargames


## Mission statement
2ITA wargames is the end goal of this project. The idea is to make this into some kind of interactive game for students where they can "fight" eachother by logging on to the pi terminal. The idea is to connect all of these to a LAN and then use network commands to discover eachother and get points by doing actions to eachother. The idea isnt fully sketched out yet, I'm experimenting to see what we can do.
